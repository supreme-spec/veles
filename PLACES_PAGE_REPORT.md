## ⚠️ ИЗВЕСТНАЯ ПРОБЛЕМА:

В файле `getAllPlacesData.ts` есть **70 дубликатов** старых вариантов названий стран:
- Старые варианты с суффиксом `-gid` (например: `kitai-gid`, `yaponiya-gid`)
- Устаревшие русские транслитерации (например: `bolgariya-gid`, `germaniya-gid`)

### Эти дубликаты НЕ ВЛИЯЮТ на работу сайта, потому что:
1. Система использует резервный механизм: если для `countryId` нет точного совпадения в `switch`, возвращается `default: return places`
2. Все современные английские названия (`china`, `japan`, `germany` и т.д.) уже есть и работают корректно

### Рекомендация:
Эти дубликаты можно удалить для чистоты кода, но это не срочно и не критично для функционала.
