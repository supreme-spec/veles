# Звуковые файлы для ParallaxCard

## Необходимые звуки:

1. **city-ambience.mp3** - Атмосфера европейских городов (шум улиц, голоса людей)
2. **jungle-birds.mp3** - Джунгли Азии (птичьи трели, шорохи листвы)  
3. **ocean-waves.mp3** - Океанские волны и прибой
4. **rainforest.mp3** - Южноамериканские тропики (дождь, птицы, насекомые)
5. **savanna-wildlife.mp3** - Африканская саванна (львы, слоны, птицы)
6. **bushland.mp3** - Австралийская бушленд (ветер, птицы, кустарники)

## Требования:
- Формат: MP3
- Длительность: 10-30 секунд (будет зацикливаться)
- Громкость: Очень тихая (0.05 от максимальной)
- Качество: 128-192 kbps
- Размер: до 500KB каждый файл

## Источники:
- freesound.org
- zapsplat.com  
- pixabay.com/audio
- Adobe Audition для редактирования

## Психологический эффект:
Эти звуки активируют лимбическую систему мозга, создавая эмоциональную связь с направлением путешествия еще до клика.
