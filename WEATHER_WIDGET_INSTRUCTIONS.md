# Инструкция по активации виджета погоды OpenWeatherMap

## 🚀 Быстрый старт (демонстрационный режим)

Виджет уже работает в демонстрационном режиме с тестовыми данными.

## 🔧 Активация реальной погоды

### Шаг 1: Получение API ключа
1. Перейдите на [OpenWeatherMap](https://openweathermap.org/api)
2. Зарегистрируйтесь (бесплатно)
3. Получите бесплатный API ключ (Free tier: 1,000 вызовов/день)

### Шаг 2: Добавление ключа в переменные окружения
Создайте или обновите файл `.env.local` в корне проекта:

```env
NEXT_PUBLIC_OPENWEATHER_API_KEY=ваш_ключ_api_здесь
```

### Шаг 3: Перезапуск приложения
```bash
npm run dev
```

## ✅ Что будет работать

После активации API ключа виджет будет показывать:
- 🌡️ Текущую температуру в Алжире
- ☁️ Актуальные погодные условия
- 💧 Влажность воздуха
- 💨 Скорость ветра
- 📊 Атмосферное давление
- ⏰ Дата и время последнего обновления

## 🎯 Особенности реализации

- **Язык**: Русский (автоматически)
- **Единицы измерения**: Метрические (°C, м/с)
- **Обновление**: Автоматическое при загрузке страницы
- **Дизайн**: Адаптивный, темный/светлый режим
- **Fallback**: Демо-данные при отсутствии API ключа

## 📱 Мобильная оптимизация

Виджет полностью адаптирован для мобильных устройств и планшетов.

## 🔒 Безопасность

API ключ хранится в переменных окружения и не попадает в клиентский код напрямую.
